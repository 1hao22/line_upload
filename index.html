影印機使用說明

LINE

掃描QR Code

掃描螢幕上之QR Code後點選下方箭頭。

2.加入好友

勾選想要列印的檔案後點擊列印。

3.查看驗證碼

將下方紅色顯示碼輸入至LINE聊天室中。

4.輸入驗證碼

輸入機台上方驗證碼後點擊左下角「+」。

5.點選檔案

點選檔案。

6.點選上傳檔案

點選要列印的檔案後點擊右上角「完成」上傳至聊天室。

7.上傳檔案

收到回傳成功訊息後回到機台。

8.收到回傳成功訊息

勾選即將列印的檔案。

9.列印設定

設定顏色、單雙面、份數及尺寸，確認後點選確定。

10.付款方式

確認金額並選擇付款方式至下方機台感應付費。

11.等待列印

付款成功後等待列印。

12.取件

至影印機取出文件，若不需列印明細即完成此次服務。
